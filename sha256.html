SHA-256 (Secure Hash Algorithm 256-bit)

Overview

SHA-256 is a part of the SHA-2 family of cryptographic hash functions, designed by the U.S. National Security Agency (NSA) and published in 2001.

Working

SHA-256 operates on 512-bit blocks of data. The input is first padded and then processed in 512-bit chunks. The algorithm uses a series of logical functions, constants, and word operations to iteratively generate a 256-bit hash.

Application

SHA-256 is used in various security applications and protocols, including TLS, SSL, PGP, SSH, and IPsec. It's also used in blockchain technology, including Bitcoin, for verifying transactions.

Time Complexity: O(N) For a message of N bits, the number of 512-bit blocks is N/512. The overall complexity is the product of the number of blocks and the complexity per block.

Space Complexity: O(1) Includes message schedule storage, working variables, and intermediate hash values.

Example

Input: "Hello, World!"

SHA-256 Hash: dffd6021bb2bd5b0af676290809ec3a53191dd81c7f70a4b28688a362182986f
